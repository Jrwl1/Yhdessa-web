Assets generated from provided logo screenshot.

Files:
- badge-*.png : circular badge with transparent outside.
- logo-*.png  : logo content only, white background removed (transparent). Best for placing on colored backgrounds.
- square-*.png: square crop with original white background.
- favicon.ico : multi-size favicon.

Suggested usage:
- favicon.ico
- apple-touch-icon.png -> use badge-180.png
- android-chrome-192x192.png -> badge-192.png
- android-chrome-512x512.png -> badge-512.png
